A OCR app that can recognize texts on image. 

This App is based on <a href="https://github.com/tesseract-ocr/tesseract/blob/master/README.md">Tesseract 4 </a> and its is first of its kind which is based on Tesseract 4. This app is made possible by a library [Tesseract4Android](https://github.com/adaptech-cz/Tesseract4Android).


## ðŸš€<font color="#2467d6"><b>Features</b></font>
* Extract Text From Images.
* Copy data to Clipboard.
* Select any part of Text.
* Based on latest [Training Data](https://github.com/tesseract-ocr/tessdoc/blob/master/Data-Files.md).
* Recognized 102  [languages](https://tesseract-ocr.github.io/tessdoc/Data-Files)
* 3 Type of data i.e 'Best', 'fast', 'standard' for more info [see](https://github.com/tesseract-ocr/tessdoc/blob/master/Data-Files.md)
* Recognize Maths/equation detection

## <font color="#2467d6"><b>Required Permissions </b></font>

* External Storage: For storing the training Data
* Internet : Downloading the training data 
* Access Internet state: For proper downloading of training data.
* Note: None of your data is send to anywhere
